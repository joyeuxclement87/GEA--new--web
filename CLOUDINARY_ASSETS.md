## Setup

1. Create/use a Cloudinary account and note your **cloud name**.
2. Set `NEXT_PUBLIC_CLOUDINARY_CLOUD_NAME=<your-cloud-name>` in your `.env.local`.
3. Upload each image below to Cloudinary using the **exact Public ID** shown (flat public IDs, no folders — e.g. `heroes-services-mep-engineering`).
4. Once uploaded, the corresponding page will automatically pick up the image — no code changes needed.

A preview/delivery URL follows the pattern: `https://res.cloudinary.com/<your-cloud-name>/image/upload/f_auto,q_auto,w_<width>,c_fill/<public-id>`
